# Skill 模板（官方规范版）

可直接复制使用，替换 `{占位符}` 为实际内容。

---

```yaml
---
# ================================================================
# 标识
# ================================================================
name: {skill-name}
description: |
  {一句话描述核心功能}。当用户提到"{关键词1}"、"{关键词2}"、
  "{关键词3}"时触发。
argument-hint: "[{参数说明}]"

# ================================================================
# 触发控制
# ================================================================
disable-model-invocation: false
user-invocable: true
# paths: "*.md, src/**/*.ts"

# ================================================================
# 执行模式
# ================================================================
context: fork
agent: "general-purpose"
allowed-tools: Read Write Edit Glob Grep
# model: sonnet
# effort: medium
---
```
